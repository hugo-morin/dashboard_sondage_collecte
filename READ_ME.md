
# Dashboard — Sondage matières résiduelles

Dashboard interactif (Streamlit) pour l’analyse des résultats du sondage
sur la collecte des matières résiduelles à Entrelacs.

Les réponses proviennent d’un sondage volontaire et sont présentées à des fins de synthèse. Les propos de réponses ouvertes ont été regroupés par thèmes afin de refléter les préoccupations générales exprimées.

## Méthodologie
- Questions fermées : statistiques descriptives (%)
- Questions ouvertes : synthèse thématique par regroupement d’idées
- Filtre : Résident·es / Villégiateur·es
